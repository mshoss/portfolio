# Mohammad Shahadat Hossain Portfolio

## Important
Do not open the site from inside the ZIP preview.

1. Right-click the ZIP
2. Click "Extract All"
3. Open the extracted folder
4. Double-click `index.html`

## Structure
- `index.html`
- `assets/profile.jpg`
- `documents/resume.pdf`

## Deploy
Upload all three items to GitHub:
- index.html
- assets folder
- documents folder

Then deploy the GitHub repository on Vercel.
